# 💰 Pocket Tracker

Pocket Tracker is a Flutter-based personal expense management application that helps users record, categorize, and analyze their daily financial transactions.

## ✨ Features

* Firebase Authentication
* Add Income & Expenses
* Edit & Delete Transactions
* Monthly Budget Management
* Expense Statistics
* Category-wise Analysis
* CSV Export
* Clean Dashboard UI
* Profile Screen
* Dark mode

## 🛠️ Technologies Used

* Flutter
* Dart
* Firebase Authentication
* Cloud Firestore
* FL Chart
* Provider

## 📱 Screens

* Splash Screen
* Login & Signup
* Dashboard
* Add Transaction
* Statistics
* Budget
* Profile
